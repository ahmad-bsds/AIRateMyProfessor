Concept, Design, and Development.

Challenge

Finding top professors in the idustry.

Goal

Create an application that helps people to find their favourite professor from the given data.

Result

Student are now able to find the best professor.

Design and Architecture

This project consists of Pinecone vector DB, Google Gen API, Python, and Flask application.

Implementation

Get data in a JSON file then store it in the Pinecone DB. Use Langchain and Google-Gen api to chat with retrieved data. Use Flask to make it accesible to the students. Note:The builtin data of professors is temporary and generated from LLM.

Future Enhancements

Allow students to send professors data from application so that they can chat with their professors who are not in the DB. OR Scrape professors data from web.
